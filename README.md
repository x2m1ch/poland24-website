## Начало работы

### Установка git
Установите git: https://git-scm.com/install/

### Копирование проекта 
Выполните команду в терминале: 
```sh 
$ git clone https://github.com/x2m1ch/poland24-website.git
 ```

### Установка node js
Установите node js: https://nodejs.org/en/download

### Переход в папку проекта 
Зайдите в терминал и с помощью команды ``` cd ``` перейдите в склонированную папку

Пример для Windows:

``` sh 
$ cd ВашДиск:\Users\ВашеИмяПользователя\ПутьДоПапки
```

Примечание: каждую папку разделяет ```\```

### Инициализация всех пакетов
Зайдите в папку проекта через терминал и выполните:
```sh
$ npm install
```

### Запуск
Выполните в терминале
``` sh
$ npm run dev
```
